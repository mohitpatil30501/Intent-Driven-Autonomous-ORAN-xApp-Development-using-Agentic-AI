#O-RAN.WG6.TS.O2-IM
The O2 Information Model provides the information and data models for the O2 IMS interface.

## Information Model
The technical specification is an O-RAN formatted specification which provides a model for several namespaces. Each namespace identifies the classes and their relationships within it. This information model is found in the contained word document.

## Data Model
The Data Model is done in yaml files following the Open API Specification (OAS) format. Each namespace has its own yaml file. The yaml files can be found in the resources subdirectory.
